Проверь текущий статус спринта:

1. Прочитай @docs/SPRINT_PLAN.md
2. Определи текущий активный спринт
3. Перечисли завершённые задачи ([ ] -> [x])
4. Перечисли оставшиеся задачи
5. Оцени прогресс в процентах
6. Предложи следующие шаги

Формат вывода:
- Sprint: N — название
- Progress: X/Y задач (Z%)
- Next action: конкретное действие
